Input and Output

In this example, there is a form with an input field where the user can enter their name. When the "Submit" button is clicked, the displayOutput() JavaScript function is called. It retrieves the input value, and the output is displayed in the
element with the id "output". Save the above code as a .html file and open it in a web browser. You will see a simple webpage with an input field. When you enter your name and click "Submit," it will display a greeting message with your name. Feel free to customize this example and add more functionality to suit your specific needs.
